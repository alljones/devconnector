# DevConnector (MERN)

Small social network with authentication, profiles, dashboards and posts.

### Technologies Used:

- Javascript (ES6+)
- Node.js/Express
- MongoDB
- React/Redux
- Twitter Bootstrap 4
- Heroku & Git for Deployment
